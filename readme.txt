A plugin to display posts in a widget with title, thumb, excerpt, date and author.

== Description ==

Simple SEO Categories Posts is a plugin/widget, for displaying on your site the posts from specified categories.

= Widget options =

* Title
* Categories IDs
* Number of posts
* Display Title
* Display Thumb
* Thumb Width
* Thumb Height
* Thumb Crop
* Display Excerpt
* Excerpt Length
* Display Date
* Display Author
* Before Author
* Order By
* Normal/Reverse
* Title Order
* Thumb Order
* Excerpt Order
* Date Order
* Author Order

[Detailed description and guide](https://www.hostdog.gr/en/blog/simple-seo-categories-posts-widget-wordpress)

== Installation ==

1. Upload the entire `simple-seo-categories-posts` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
